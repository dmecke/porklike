Controls

Move with the arrow keys. Use [x] to select / confirm and [z] to cancel.

This is a javascript port of the PICO-8 game created in Lazy Devs YouTube Tutorial. I tried to stay as close to the original code as possible, though some parts had to be adjusted due to the platform differences. You can find the source code at GitHub and me at Twitter.
